Course overview

This course helps you obtain numerical solutions to problems in engineering and science. It introduces MATLAB as both an interactive computing system and a programming language, then builds the core techniques of numerical methods on top of it.

Topics covered: basic root-finding procedures; fixed-point iteration; bisection; Newton's method; roots of polynomials; solving systems of equations; least-squares fitting of a curve to data; interpolation and polynomial approximation; and numerical solutions of ordinary differential equations.

Prerequisites: BCS-102, MTH-114 2 hrs Lecture + 2 hrs Lab Textbook: Chapra, 5th ed.

How to use this site

📖 Read the lessons

Each module reads top to bottom. Theory comes first, then a worked example with the exact numbers you'd compute by hand, then the MATLAB implementation.

🎛️ Play with the demos

Drag the sliders and press the step buttons. Watching bisection bracket a root or Euler's method drift away from the true curve builds intuition no equation can.

⌨️ Edit the code

Code blocks marked editable can be changed and copied straight into MATLAB. Try a different function or tolerance and run it yourself.

✅ Test yourself

Finish each module with a quiz aligned to its CLO, then mark the module complete. Your progress bar (left) is saved in this browser.
